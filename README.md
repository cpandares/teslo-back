# Teslo Shop Backend


# Clonar repositorio e instalar dependencias
```bash
  npm install
```

# Crear archivo de variables de entorno
```bash
  cp .env.example .env
```

# Crear base de datos
```bash
  docker-compose up -d
```

# En modo desarrollo
```bash
  npm run start:dev
```

# Ejecutar Seed de la base datos
```bash
  localhost:3000/api/seed
```
